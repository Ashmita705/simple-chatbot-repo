# Simple Python Chatbot 
A basic rule-based chatbot built using Python that responds to user inputs using simple if-else logic.

## Features
- Greets the user
- Answers basic questions like "how are you" and "what is your name"
- Exits on "bye"

## Tech Used
- Python
